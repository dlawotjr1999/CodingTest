### 3.2. Greedy Algorithm

### 3.3 Dynamic Programming
- DP 배열이 무엇인지 선언하는 방법
    - 최댓값 / 최솟값 구하기
    - 최소 횟수 / 최소 연산
    - 가능 여부
    - 방법의 수
⇒ **‘부분 문제 상태’가 바로 DP 배열에 들어갈 값**
